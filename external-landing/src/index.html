Trying to access the Huggins Cloud?

Let's get you conencted. If you don't have WireGuard set up already, let's take care of that first.

  1. Download and install WireGuard for your device
  2. Open the application and import the provided VPN profile
  3. Accept any prompts to install VPN configurations that may appear
  4. Enable the VPN connection

You should now be connected! You should see the application dashboard next time you visit this page.
